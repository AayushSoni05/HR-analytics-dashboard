# 📊 HR Analytics Dashboard

## 📌 Overview
This project presents an interactive Power BI dashboard to analyze HR data, focusing on salary trends, department performance, and workforce insights.

## 🛠️ Tools Used
- Power BI

## 📊 Key Features
- Department-wise salary distribution  
- Monthly salary trend analysis  
- Average and total salary KPIs  
- Interactive filters for dynamic analysis  

## 📈 Key Insights
- IT department shows the highest salary expenditure  
- Salary trends vary across different months  
- HR department has comparatively lower average salary  

## ▶️ How to Use
1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Explore dashboard using filters and visuals
